# NovaWing Gameplay Ideas

The fastest way to make NovaWing more fun is to add more risk/reward decisions. The current game already has a strong arcade base: shooting, boost, weapon upgrades, obstacles, a boss fight, stats, and leaderboard hooks. These ideas build on those existing systems.

## 1. Make Boost More Than Speed

Boost already affects player speed, level progress, and world speed. Add rewards for dangerous boost play:

- Grazing bullets while boosting gives points or boost refill.
- Boosting through enemy explosions extends a combo.
- Optional boost ram destroys small enemies but costs energy.
- Near-miss text such as `GRAZE +25` gives immediate feedback.

## 2. Add A Combo Meter

Enemy kills currently award a flat score. A combo system would make aggressive play more exciting:

- Quick kills increase a score multiplier.
- Taking damage resets the combo.
- Boost kills or grazes can extend the combo timer.
- Combo milestones can trigger stronger effects or bonus points.

## 3. Replace Pure Random Enemies With Wave Patterns

Enemy spawns currently pick random lanes. Authored wave patterns would make the game feel more skill-based:

- Three enemies in a diagonal.
- Two interceptors from opposite lanes.
- Asteroid wall with one safe gap.
- Slow enemy followed by a fast chaser.

Readable patterns are usually more satisfying than random traffic because players can learn, react, and improve.

## 4. Make Powerups Create Choices

Powerups currently upgrade the weapon and then become score. Add different pickup types:

- Spread shot.
- Laser.
- Shield.
- Repair.
- Bomb.
- Temporary overdrive.

Even two or three pickup types would make collecting powerups feel more exciting.

## 5. Give The Boss Phases

The boss currently uses a repeated missile volley pattern. Add phase changes at health thresholds:

- Phase 1: missiles.
- Phase 2: missiles plus drone adds.
- Phase 3: warning laser lanes or faster volleys.

Boss fights feel better when the player can sense escalation.

## 6. Add More Game Feel

Small presentation changes can make the same mechanics feel much better:

- Screen shake on larger explosions.
- Brief hit-stop when killing enemies.
- Bigger explosions on combo milestones.
- Engine pitch rising during boost.
- Warning flash before boss attacks.

## Best First Changes

If only three features are added first, prioritize:

1. Combo meter.
2. Boost grazing.
3. Wave patterns.

Those should make the existing game feel more skill-based without requiring a full rewrite.
